SUDOKU
Es juego de lódica y rompecabeza diseñado para ejercitar la mente, la concetración y el pensamiento analítico.
Puedes pedir la opción de ayuda para relevar un número aleatorio, pero solo hasta el penúltimo número oculto.

Estructura MVC
Model: CellPosition, SudokuBoard, SudokuPuzzle, SudokuRule, SudokuRuleAdapter.
Controller: contiene el controlador de la pantalla del juego(Game Controller)
Resources: archivos fxml para la IU del juego.
